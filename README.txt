Це навчальний багатосторінковий сайт.
Сторінки:
1) index.html — Головна сторінка (привітання, логотип, список розділів).
2) about.html — Про автора (опис у абзацах, список захоплень, блок aside).
3) gallery.html — Галерея (зображення у figure + figcaption, список категорій).
4) contacts.html — Контакти (способи зв’язку, список посилань).
Усі сторінки мають:
- header з навігацією
- main з основним контентом
- footer з автором і роком
Використані теги: div, p, span, i, em, b, strong, u, a, img, ul, ol, li, nav, h1–h6,
header, main, footer, section, article, aside, figure, figcaption.
